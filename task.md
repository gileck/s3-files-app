# Main Feautres:

* The MONGO URI is hard coded in the .env file (MONGO_URI)
* Select Collection from collection list (top of the dashboard)
* view all collection's documents in a table (rows are documents, columns are fields)
* view full document 
* Edit fields
* Run queries
